# block elements
eg p, h1-h6, div, html semantics(section, aside, main, footer, header), ul.

To convert block elements to inline elements, use
     display:inline or
     display:inline-block; 

# inline elements
eg a , span

to convert inline elements to block, use
      display:block;

# positioning
position:fixed;
absolute, relative
left, right, top, buttom.
